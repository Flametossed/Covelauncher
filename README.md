# CoveLauncher

CoveLauncher is a modern, lightweight, and elegant Game Library Manager designed to help you organize and launch your game collection seamlessly.

## Features
- **Dynamic Catalogue**: Browse available games through an integrated catalogue interface.
- **Library Management**: Automatically organizes downloaded games into a sleek, sortable library grid.
- **Smart Cover Art Fetching**: Automatically fetches and saves high-quality cover art for your games.
- **Built-in Downloader**: Handles game downloads and automatic zip extraction natively within the app.
- **Beautiful UI**: Built with React, Tailwind CSS, and a premium dark mode interface for optimal aesthetics.

## Installation & Running Locally

1. Clone the repository
2. Install dependencies: `npm install`
3. Run the development server: `npm run electron:dev`

## Building

To build the executable for your OS:
```bash
npm run electron:build
```
This will output a standalone installer in the `dist` folder.
